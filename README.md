# 🔐 Authentication System (Django)

Sistema de autenticação com cadastro, login, logout, validação de dados, proteção de rotas e mensagens ao usuário utilizando Django.

---

## 🚀 Funcionalidades

* Cadastro de usuários
* Login e Logout
* Validação de dados
* Proteção de rotas com autenticação
* Mensagens de feedback ao usuário

---

## 🛠️ Tecnologias

* Python
* Django
* Bootstrap

---

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de autenticação e fluxo de usuários em aplicações web utilizando Django.

---

## ▶️ Como executar

```bash
# Clonar o repositório
git clone <seu-repositorio>

# Entrar na pasta
cd user-auth-system

# Criar ambiente virtual
python -m venv venv

# Ativar ambiente
venv\Scripts\activate

# Instalar dependências
pip install -r requirements.txt

# Rodar migrações
python manage.py migrate

# Executar servidor
python manage.py runserver
```
